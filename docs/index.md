## 推荐学习路径

1. 先建立 Android 开源框架全景与评价维度；
2. 再学习图片、网络、接口代理和数据解析等数据获取框架；
3. 继续深入 RxJava、Coroutines、Flow 和 Hilt 等线程、数据流与依赖管理能力；
4. 最后通过 Room、DataStore、WorkManager、Paging、Navigation、测试与性能工具完成综合工程实践。
